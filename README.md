# Assignment Tasks

## Task 1 – Test Case Creation

Prepared comprehensive manual test cases for the **Login** and **Signup** functionalities.

### Test Coverage

- Positive Test Scenarios
- Negative Test Scenarios
- Input Validation
- Boundary Value Testing
- Functional Testing
- UI Validation
- Basic Security Validation

The test cases are designed to provide comprehensive coverage of the Login and Signup functionalities.

---

## Task 2 – Defect Reporting

Prepared a detailed defect report for the following issue:

**Defect:** The application allows users to attempt login with an invalid email format.

The defect report includes:

- Defect Summary
- Environment Details
- Test Data
- Steps to Reproduce
- Expected Result
- Actual Result
- Severity and Priority
- Impact Analysis
- Suggested Fix
- Traceability
